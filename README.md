# Tinker's Quality
### 介绍
本模组为 ArrowMinecraft 的锻刀大赛而生，为匠魂的工具、工具部件引入了一套品质系统。 
工具和工具部件的统计数据会随着品质的改变而变化。

### 说明
本模组极度依赖Mixin。所以可预见的是，它的兼容性应该不会特别好……

### 规划
- 数据驱动：
目前的修正统计数据的办法是硬编码的，所有材料的统计数据随品质变化的程度都是一样的。
日后准备把算法改成数据驱动，可以用数据包自定义材料的品质修正方法。例如给玛玉灵增加一个额外的伤害修正项，给钢增加一个恒为正的耐久修正项。
- JEI支持：
支持用JEI查看每种材料的品质修正数据。
- KubeJS支持：
提供一些事件以供更深度更定制化的魔改。
